Proyectos Desarrollados Individualmente

Simulador de Enrutamiento Dinámico en Java basado en RIP y OSPF

Este proyecto consiste en el desarrollo de un simulador de redes en Java que implementa los protocolos de enrutamiento RIP y OSPF. El software realiza el análisis de tablas de enrutamiento, evalúa métricas de costo y permite la simulación de cambios en la topología de la red. Además, genera código de configuración para routers virtualizados y exporta datos a un entorno de bajo nivel mediante scripts en Python para su implementación en dispositivos reales.

Firewall basado en Reglas Dinámicas con Inspección Profunda de Paquetes

Este proyecto desarrolla un firewall en Java con soporte para la inspección profunda de paquetes (DPI), permitiendo el filtrado avanzado de tráfico de red. Se incluyen módulos para el análisis sintáctico y semántico de reglas de seguridad escritas en un lenguaje personalizado, que luego se traducen a configuraciones optimizadas en código ensamblador para su implementación en sistemas embebidos o entornos de bajo nivel.

Compilador de Políticas de Calidad de Servicio (QoS) en Java

Este proyecto implementa un compilador en Java para definir y gestionar políticas de calidad de servicio (QoS) en redes. Incluye análisis léxico y sintáctico de configuraciones de ancho de banda, priorización de tráfico y gestión de colas. El sistema genera código intermedio que posteriormente se traduce a scripts en Bash y configuraciones en código ensamblador para su aplicación en routers y switches de bajo nivel.
