SGFC

Es una aplicación diseñada para optimizar la producción, venta y control de inventario de fresas con crema en negocios locales. Con una interfaz sencilla y moderna, permite llevar un seguimiento en tiempo real del stock de fresas, crema y toppings, además de registrar ventas diarias y sugerir compras automáticas según el consumo promedio.

CDF

Con una interfaz amigable y funcionalidades adaptadas a las necesidades de los entrenadores y jugadores, este software permite gestionar calendarios de partidos, registrar estadísticas clave de desempeño (goles, asistencias, tarjetas), controlar la asistencia a entrenamientos y partidos

CDF

Proyectos Desarrollados Individualmente

NetMon Automation

Es una aplicación diseñada para automatizar la monitorización y gestión de redes locales. Con una interfaz intuitiva, permite realizar un seguimiento en tiempo real del tráfico de red, detectar anomalías y generar reportes automáticos sobre el rendimiento del sistema. Además, incluye funcionalidades para configurar alertas personalizadas cuando se detectan problemas como caídas de conexión o sobrecarga de servidores.

SecureAPI Gateway

Es un sistema desarrollado para garantizar la seguridad en aplicaciones web mediante la implementación de un gateway API seguro. Este proyecto permite autenticar solicitudes HTTP, validar tokens JWT y cifrar datos sensibles antes de ser transmitidos. Además, incluye un módulo de logging para registrar todas las actividades en la red, facilitando auditorías de seguridad y cumplimiento normativo.

CloudDeploy Manager

Es una herramienta de automatización diseñada para simplificar el despliegue y administración de servicios en la nube. Con esta aplicación, los usuarios pueden gestionar instancias virtuales, configurar firewalls, monitorear recursos y escalar automáticamente según la demanda. Incluye integración con Docker para facilitar el despliegue de contenedores y asegurar un entorno de trabajo eficiente y seguro.
